window git bash 增加软件 | GitBash增加linux命令

1 下载软件

  • 软件下载地址

比如: 我要安装jq命令
在下载网页中找到jq安装包, 点击下载即可

2 将二进制包放入到 git bash 的 usr/bin 目录

  • 下载安装”xx.tar.zst”安装包后, 用支持zst压缩包的解压缩软件(比如: 7zip)进行解压缩
  • 将相关解压缩出来的文件放入git bash的安装目录.
    • 解压包的文件都带有usr目录, 所以把压缩包的usr目录直接覆盖到git bash安装目录usr目录即可

把压缩包的usr文件夹放入到git bash文件夹中, 与git bash文件夹的usr文件夹合并

我的git bash安装路径是”C:\Program Files\Git”

3 重启 git bash

重新打开git bash, 测试新安装的命令

比如: 测试新安装的jq命令
